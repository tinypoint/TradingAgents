# Design Tokens

## Layout
- Base spacing: 8px system
- Section vertical spacing: `space-y-12` to `space-y-16`
- Radius: `rounded-xl`

## Palette
- Page: `bg-gray-50`
- Card: `bg-white`
- Section break: `bg-gray-100`
- Primary: `#2563EB`
- Success: green scale
- Risk: red scale

## Typography
- Headline: gray-900, semibold
- Body: gray-700, regular
- Meta: gray-500
- Weak hint: gray-400
- Body line-height: relaxed (~1.7)

## Visual Principles
- White space first, low noise
- 80% grayscale, 10% primary, 10% status colors
- Prefer hierarchy by background + shadow over heavy borders
- Charts are primary visual focus
