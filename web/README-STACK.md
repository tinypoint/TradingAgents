# Frontend Stack Rules

## Fixed Stack
- Vite + React + TypeScript
- Tailwind CSS
- shadcn/ui + Lucide
- React Router
- Zustand (light UI state only)
- Axios
- lightweight-charts
- ECharts
- React Flow
- Sonner
- Markdown: react-markdown + remark-gfm + rehype-highlight

## Keep
- remark-gfm
- rehype-highlight

## Forbidden
- TanStack Query/Table/Virtual
- Zod
- xlsx
- copy-to-clipboard
- katex
- Any second UI framework (AntD/MUI/Chakra)

## UX Constraints
- Long scroll research page, not fixed-height dashboard
- Light research platform style
- Do not introduce dark terminal style
